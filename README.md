# 步骤截图助手

一个功能强大的 Chrome 浏览器扩展插件，支持所有基于 Chromium 内核的浏览器（包括但不限于 Chrome、Edge、360 浏览器等），用于自动化截图和操作记录。

## 功能特性

### 核心功能

1.  **扩展操作页面**：点击浏览器工具栏的插件图标，显示操作界面，支持弹出窗口和侧边栏两种模式。
2.  **自动截图**：点击"开始截图"后，在页面上进行的每一次点击都会自动触发截图。
3.  **点击高亮**：点击位置会显示一个红色半透明的圆形标记，并在截图中体现（可配置开启/关闭）。
4.  **智能截图**：
   - 自动处理新标签页打开场景，确保截图正确
   - 支持下拉菜单等快速交互的准确截图
   - 可配置截图缩放比例（10%-100%）
5.  **自动下载**：截图完成后会自动下载到浏览器的默认下载目录中，按域名自动分组存储。
6.  **图片管理**：
   - 按域名自动分组显示本次录制的图片
   - 支持展开/折叠分组
   - 缩略图预览
   - 全屏预览模式
   - 图片删除功能
   - 图片复制功能（支持复制到剪贴板）
   - 快速打开文件所在文件夹

### 预览功能

- **全屏预览**：点击缩略图可全屏查看图片
- **图片切换**：
  - 左右箭头按钮切换
  - 键盘快捷键：`←` 上一张，`→` 下一张，`Esc` 关闭预览
- **图片计数器**：显示当前图片位置（如 "2 / 5"）
- **复制图片**：在预览模式下，点击复制按钮可将图片复制到剪贴板

### 配置选项

- **显示点击红色标注**：可开启/关闭点击时的红色标记显示
- **截图缩放比例**：可调整截图缩放比例（默认 60%）
- **禁用截图地址**：支持配置禁用截图的网站地址列表
  - 支持通配符匹配（`*` 和 `**`）
  - 默认禁用：腾讯文档、飞书文档等协作平台
  - 页面加载时自动检测，禁用页面会显示红色提示横幅
  - 在禁用页面上无法开始截图

## 安装说明

### Chrome 浏览器安装步骤

1.  打开 Chrome 浏览器，在地址栏输入 `chrome://extensions/` 进入扩展程序管理页面。
2.  打开页面右上角的"开发者模式"开关。
3.  点击"加载已解压的扩展程序"按钮。
4.  在弹出的文件选择对话框中，选择本项目的根目录。
5.  安装完成后，浏览器工具栏将显示插件图标。

> **提示**：本插件同样适用于其他基于 Chromium 内核的浏览器（如 Edge、360 浏览器等），安装方法类似。

## 使用说明

### 基本使用

1.  点击浏览器右上角的插件图标打开操作界面（支持弹出窗口和侧边栏模式）。
2.  点击"开始截图"按钮开始录制。
3.  在网页上进行操作（点击任意位置）。
4.  每次点击都会自动下载一张包含点击位置红点的截图。
5.  再次点击"停止截图"按钮即可结束录制。

### 配置选项

- **显示点击红色标注**：勾选后，点击时会显示红色圆形标记；取消勾选则不显示标记。
- **截图缩放比例**：调整截图的大小比例，范围 10%-100%，默认 60%。
- **禁用截图地址**：
  - 支持配置禁用截图的网站地址列表
  - 支持通配符：`*` 匹配除 `/` 外的任意字符，`**` 匹配任意字符（包括 `/`）
  - 示例：
    - `docs.qq.com/sheet/**` - 禁用腾讯文档表格的所有路径
    - `*.example.com/**` - 禁用 example.com 所有子域名的所有路径
    - `**feishu.cn/wiki/**` - 禁用所有 feishu.cn 子域名下的 wiki 路径
  - 配置方式：在配置区域输入地址后按回车或点击 `+` 按钮添加
  - 支持编辑和删除已配置的地址
  - 禁用页面会在页面顶部显示红色提示横幅
  - 在禁用页面上点击"开始截图"会被阻止并提示

### 图片管理

- **查看图片列表**：停止录制后，可在操作界面查看本次录制的所有图片。
- **按域名分组**：图片自动按网站域名分组显示，点击分组标题可展开/折叠。
- **预览图片**：
  - 点击缩略图可全屏预览
  - 使用左右箭头按钮或键盘 `←` `→` 键切换图片
  - 按 `Esc` 键或点击背景关闭预览
  - 支持复制图片到剪贴板
- **删除图片**：
  - 鼠标悬停在图片列表项上，点击删除按钮可删除图片
  - 在预览模式下，点击删除按钮也可删除当前图片
  - 删除时会显示确认对话框，避免误操作
- **打开文件夹**：点击分组标题旁的文件夹图标，可快速打开该域名图片所在的文件夹。

### 快捷键

在图片预览模式下：
- `←` 键：切换到上一张图片
- `→` 键：切换到下一张图片
- `Esc` 键：关闭预览

# 步骤截图助手

一个功能强大的 Chrome 浏览器扩展插件，支持所有基于 Chromium 内核的浏览器（包括但不限于 Chrome、Edge、360 浏览器等），用于自动化截图和操作记录。

## 功能特性

1.  **扩展操作页面**：点击浏览器工具栏的插件图标，即可打开操作界面，提供"开始截图"和"停止截图"功能按钮。
2.  **自动截图**：启动截图功能后，在网页上的每一次点击操作都会自动触发截图，方便记录操作步骤。
3.  **点击高亮**：点击位置会实时显示一个红色半透明的圆形标记，并在截图中清晰体现，便于识别操作位置。
4.  **自动下载**：截图完成后会自动保存到浏览器的默认下载目录，文件名采用时间戳命名，方便管理和查找。

## 安装说明

### Chrome 浏览器安装步骤

1.  打开 Chrome 浏览器，在地址栏输入 `chrome://extensions/` 进入扩展程序管理页面。
2.  打开页面右上角的"开发者模式"开关。
3.  点击"加载已解压的扩展程序"按钮。
4.  在弹出的文件选择对话框中，选择本项目的根目录。
5.  安装完成后，浏览器工具栏将显示插件图标。

> **提示**：本插件同样适用于其他基于 Chromium 内核的浏览器（如 Edge、360 浏览器等），安装方法类似。

## 使用说明

1.  点击浏览器右上角工具栏中的插件图标，打开操作弹窗。
2.  点击弹窗中的"开始截图"按钮，启动自动截图功能。
3.  在网页上进行正常的操作（如点击按钮、链接等）。
4.  每次点击操作都会自动触发截图，截图会包含点击位置的红色标记，并自动下载保存。
5.  完成操作后，再次点击插件图标，在弹窗中点击"停止截图"按钮即可结束录制。
